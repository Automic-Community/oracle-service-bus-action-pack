    ...    -${Options}: This field specifies options to use like su or sudo.
    ...    -${SudoUsername}: This field specifies username for the super user for unix environment.
    ...    -${WeblogicHome}: This field specifies system environment variable for Weblogic.
    ...    -${JavaOptions}: This field specifies the java options.
    ...    -${OSBHome}: This field specifies system environment variable for OSB.
    ...    -${Protocol}: This field specifies protocol over which you want to connect to OSB.
    ...    -${Hostname}: This field specifies hostname of OSB.
    ...    -${Username}: This field specifies username for OSB.
    ...    -${Password}: This field specifies the password for provided username.
    ...    -${Port}: This field specifies port on which OSB is running.
    ...    -${SessionName}: This field specifies name of the session you want to create.
